# Лабораторные работы по Java

## Структура репозитория

- **lab1-maven/** - Лабораторная работа №1: Maven проект с наследованием, логированием и JSON сериализацией
- **lab2-gradle/** - Лабораторная работа №2: Gradle проект с кастомными задачами и билд-информацией
- **lab3-spring/** - Лабораторная работа №3: Разработка веб-приложения на Spring Boot. Введение в IoC/DI
